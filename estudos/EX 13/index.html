utilizando audios

voce pode adicionar um audio em html5 utilizando a tag "audio" e adicionando o audio a ser utilizado no parametro "src", o parametro "autplay" serve para tocar a musica automaticamente, o parametro "controls" serve para aparecer a barrinha de inicio e duracao do audio detalhe importante, durante os testes efetuados neste dia, a tag "autoplay" e "controls" só funciona se for colocada antes do parametro "src"

possa ser que o navegador que esta sendo criado o site nao venha ter compatibilidade com o formato de audio que esta sendo utilizado, os formato de audios que existem sao : MP3 WAV E OGG, caso nao houver compatibilidade utilize de outro formato de som !!!

tambem pode ser utilizado outra maneira de adicionar audios e ja deixando programado diversos audios de formataçoes diferentes para caso um nao funcionar o navegador teste os demais, isso é utilizado e feito dentro da tag "audio" e o parametro a ser utilizado nao é o "src" que vem como padrao da tag mas sim o "source" como parametro "src"este parametro é onde é adicionado no codigo o audio e parametro "type" este parametro é para adicionar ao codigo o formato do audio

existem os parametros "controls" "loop" e "autopay" que pode ser adicionado dentro da tag audio, este é o siginicado de cada um deles :

  • controls
    • serve para deixar o visual de inicio e duracao do audio
  • loop
    • serve para a musica ficar repetindo infinitamento pelo site
  • autoplay
    • serve para tocar musica automaticamento ao abrir o site
