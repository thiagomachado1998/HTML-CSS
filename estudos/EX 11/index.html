pra adicionar um link no html é so usar a tag "a" e colocar o link dentro da opcao "href"

para abrir o link em uma nova aba use a funçao : "target _blank"

a funçao "target _self" serve para mostrar ao google que o link pertence ao mesmo site

a funcao "rel external" serve para mostrar ao google que o link é de um site externo

a funçao "rel next" serve para mostrar ao google que o link é a proxima pagina do site

a funçao "rel prev" serve pra mostrar que o link é de uma pagina anterior do proprio site

metodo para deixar link para dowload

  • arquivo zip
  • arquivo pdf

nos dois exemplos acima esta dois links de dowload

  • o primeiro link esta programado para receber um arquivo em zip entao voce adicionará a tag "a" para adicionar o link, e o parametro "download = e colocar nome do arquivo e o "type = " que vai determinar qual o tipo de documento está para download que nesse caso é "application/zip"
  • o segundo link deste exemplo segue o mesmo padrao do primeiro, só que esta programado para fazer dowload de um arquivo PDF, entao voce em relacao ao primeiro exemplo só mude o nome do arquivo e no "type=" coloque "application/pdf"

detalhe, voce tem que deixar o documento de download dentro da pasta do arquivo html do site
